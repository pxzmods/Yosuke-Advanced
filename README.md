**How to install Yosuke bot on Termux🎭**

1° comando: 

```
pkg install nodejs -y
```

2° comando:
```
pkg update && pkg upgrade -y
```

3° Comando:
```
pkg install git -y
```

4° Comando:
```
termux-setup-storage
```

5° Comando:
```
cd /sdcard/
```

6° comando:
```
git clone https://github.com/pxzmods/Yosuke-Advanced.git
```

7° Comando:
```
cd yosuke-advanced
```

**Comando para iniciar a bot🔐**
```
sh start.sh
```
